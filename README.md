# Minecraft-Mod-Searcher---SQL-project

An offline library to add and search for Minecraft mods information, can be upgraded into an online library or to automatically install mods.

This project was created as a school group project to explore practical of the language of SQL.

# Platform:

Windows 64-bit

# How to run

1. Open the folder
   
2. Run Minecraft Mod Database Searcher.exe
